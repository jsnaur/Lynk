selection');\n     return;\n   }\n   `\n\n4. **Handle loading states**\n `tsx\n   {loading ? <Spinner /> : <Content />}\n   `\n\n5. **Show feedback to users**\n `tsx\n   showSuccess('Updated!');\n   showError('Failed to update');\n   `\n\n---\n\n## Common Mistakes to Avoid\n\n❌ **Don't put backend calls in component body**\n`tsx\n// BAD\nconst MyComponent = () => {\n  const data = fetchData(); // = async () => {\n  const result = await apiCall();\n  updateUI(result);\n}\n\n// GOOD\nconst handlePress = async () => {\n  try {\n    const result = await apiCall();\n    updateUI(result);\n  } catch (error) {\n    showError(error.message);\n  }\n}\n`\n\n❌ **Don't mix too many concerns**\n`tsx\n// BAD - Too much in one component\nconst Component = () => {\n  // State management\n  // Form logic\n
